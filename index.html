ICC Cricket World Cup 2019 — Performance Analytics Dashboard

An interactive Tableau dashboard that summarizes team performance, venue trends, innings scoring patterns, and match strategy insights from the 2019 tournament.

View on Tableau Public Planning (Mural) GitHub LinkedIn

Dashboard

Use the filters inside the dashboard to explore team, venue, innings, and result-based insights.

Tip: If the dashboard appears blank, refresh once. Tableau Public sometimes takes a moment to initialize.

What this dashboard answers

  • Which teams won the most matches and achieved the best win %
  • Which venues hosted the most matches and produced high average scores
  • How innings (1st vs 2nd) impacted scoring patterns
  • How toss decisions (bat vs field) related to match outcomes
  • Chasing success rate and defending performance above 300 runs
